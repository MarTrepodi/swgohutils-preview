Open-source tooling for Galaxy of Heroes.

A self-hosted toolkit for working with SWGOH game data. It starts with comlink, the API client you run yourself — everything else builds on top.

Start with comlink View all 14 repos
~/swgoh-comlink
# run the game-API client yourself
$ docker run -d -p 3200:3200 \
    --name comlink \
    ghcr.io/swgoh-utils/swgoh-comlink
→ comlink listening on :3200

# then talk to it from python
>>> from swgoh_comlink import Comlink
>>> Comlink().get_player("ally_code")
01 / host

Run comlink

Deploy the self-hosted client via Docker or a prebuilt binary for Windows, macOS, or Linux. It speaks to the game's servers so you don't have to.

02 / connect

Pick a wrapper

Use the Python or JavaScript client libraries to query comlink from your own bots, sites, and pipelines with a clean interface.

03 / build

Add the toolkit

Layer on stat calculators, the maintained game-data mirror, and the asset extractor to turn raw responses into real features.

The toolkit

Fourteen repositories, all open source. Here are the ones to know.

Core
swgoh-comlink start here
The self-hosted API client at the center of the ecosystem. Shipped as a Docker image and as binaries for major platforms.
Shell35MIT
deploy-swgoh-comlink
Deployment recipes and container configuration to get a comlink instance up and running quickly.
Dockerfile519 forks
Client libraries
comlink-python
Python wrapper for swgoh-comlink. Idiomatic access to the client for scripts, services, and data pipelines.
Python5MIT
comlink-js
JavaScript / Node wrapper for swgoh-comlink. Drop it into web apps and Discord bots.
JavaScript2MIT
Stats & data
swgoh-stat-calc
JS package for calculating exact unit stats from roster data — the maintained fork of Crinolo's calculator.
JavaScriptMIT
swgoh-stats
A service wrapping the stat calculator so you can compute unit stats over HTTP without hosting the math.
JavaScript5MIT
gamedata
A continuously refreshed mirror of parsed game data: units, gear, abilities, and more.
Data7GPL-3.0
swgoh-ae2
Asset Extractor — download and extract the game's 2D textures for use in your own tools.
C#5MIT
